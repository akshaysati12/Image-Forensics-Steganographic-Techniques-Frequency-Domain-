# 🖼️ Image Forensics – Steganographic Techniques (Frequency Domain)

## 📌 Project Overview

This project focuses on **Image Forensics using Steganographic Techniques in the Frequency Domain**.
The goal of the project is to understand how hidden information can be embedded and detected inside images using frequency-based transformations.

In this project, different **frequency domain techniques** such as **DCT, DWT, and FFT** are explored to analyze how data can be hidden within images while maintaining image quality.

---

## 🎯 Objectives

* Understand the concept of **image steganography**
* Explore **frequency domain techniques** used in image processing
* Implement and analyze **DCT, DWT, and FFT based methods**
* Study how hidden data can be embedded and extracted from images
* Learn the role of **image forensics in detecting hidden information**

## 🔬 Techniques Implemented

### 📌 Discrete Cosine Transform (DCT)

DCT is widely used in image compression (such as JPEG).
In this project, DCT is used to transform image data into frequency components where hidden data can be embedded with minimal visual distortion.

### 📌 Discrete Wavelet Transform (DWT)

DWT analyzes images at different frequency bands with different resolutions.
It helps in embedding hidden data in specific wavelet coefficients while preserving the image quality.

### 📌 Fast Fourier Transform (FFT)

FFT converts spatial domain image data into frequency domain representation.
This technique helps in analyzing frequency components and understanding how information can be manipulated within images.
